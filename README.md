# kattis

> scuffed Kattis solutions probably in C++ 

## System requirements

- A C++ compiler such as G++ (should come with GCC)

## Development

- edit the `main.cpp` files in each folder
- run `g++` on them
- if no arguemnts, run `./a.out` to run the program
